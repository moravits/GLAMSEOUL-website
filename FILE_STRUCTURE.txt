🎯 주요 파일 설명
===============

index.html
----------
- 웹사이트의 메인 파일
- 7개 섹션 포함: Home, About, Business, Brands, Products, Network, Contact
- 이 파일만 열어도 전체 웹사이트 확인 가능

css/style.css
-------------
- 메인 디자인 스타일시트
- 색상, 폰트, 레이아웃 정의
- 애니메이션 효과 포함
- 약 1,500줄의 CSS 코드

css/responsive.css
------------------
- 반응형 디자인 처리
- 모바일, 태블릿, 데스크톱 최적화
- 브레이크포인트: 1024px, 768px, 576px

js/main.js
----------
- 네비게이션 기능
- 스크롤 애니메이션
- 카운터 효과
- 모바일 메뉴 토글
- 약 400줄의 JavaScript 코드


🔧 수정 가능한 항목
==================

쉬운 수정 (초보자)
-----------------
✏️ 텍스트 내용 → index.html
✏️ 이메일 주소 → index.html (Contact 섹션)
✏️ SNS 링크 → index.html (Footer)
✏️ 스토어 링크 → index.html (Contact 섹션)
🖼️ 이미지 교체 → images/ 폴더

중간 수정 (경험자)
-----------------
🎨 색상 변경 → css/style.css (:root 섹션)
📏 레이아웃 조정 → css/style.css
📱 반응형 조정 → css/responsive.css

고급 수정 (개발자)
-----------------
⚙️ 기능 추가 → js/main.js
🔌 API 연동 → 새 파일 추가
🗄️ 백엔드 연결 → 서버 사이드 코드 필요
